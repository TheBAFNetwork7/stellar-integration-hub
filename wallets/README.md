# Billeteras e integración

Esta sección explica **cómo conectar una app** con billeteras del ecosistema Stellar.

## Opciones
- **Freighter** (extensión de navegador; firma transacciones y Soroban).
- **xBull** (extensión; open source).
- **Kits multi-wallet** (una librería para hablar con varias billeteras con la misma API).

> Requisitos comunes: sirve tu app por **HTTPS** en desarrollo para que las extensiones funcionen correctamente.
