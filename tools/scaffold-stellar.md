# Scaffold Stellar

Toolkit para crear dApps y contratos Soroban rápido (CLI + plantillas + frontend).
- Sitio: https://scaffoldstellar.org/
- Docs: https://developers.stellar.org/docs/build/soroban-dapps/scaffold-stellar

## Instalación rápida
```bash
cargo install --locked stellar-scaffold-cli
stellar scaffold init my-dapp
cd my-dapp && npm install
```
