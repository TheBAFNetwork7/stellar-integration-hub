# Contribuir

## Tipos de contribución
- **Nueva API**: añade entrada en `/apis/registry.yml` + doc en `/apis/<nombre>.md` + ejemplo/quickstart.
- **QuickStart**: guía de 5–10 min reproducible.
- **Herramienta**: docs en `/tools`.

## Requisitos mínimos
- Sandbox gratuito o de prueba.
- Documentación pública (ideal: OpenAPI).
- Ejemplo que funcione con claves de prueba.

## Estilo de guías
- Pasos numerados, comandos copiables, duración objetivo ≤10 min.
- Sección “Solución de problemas”.
