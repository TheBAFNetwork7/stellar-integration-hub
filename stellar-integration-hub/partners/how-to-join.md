# Cómo sumar tu API al Integration Hub

## Requisitos
- **Sandbox** o plan de prueba.
- Enlace a documentación (ideal: **OpenAPI 3.0**).
- QuickStart de 5–10 min con claves de prueba.

## Qué enviar en tu PR
1. Entrada en `/apis/registry.yml`.
2. Archivo `/apis/<tu-api>.md` con:
   - ¿Qué resuelve? (1-2 líneas)
   - Autenticación (API key / OAuth)
   - Enlace a sandbox y límites de rate
   - Snippet minimal de uso
3. Ejemplo o QuickStart que compile/corra.

## SLA durante la hackathon
- Contacto (Slack/Discord/Email)
- Horario de soporte
