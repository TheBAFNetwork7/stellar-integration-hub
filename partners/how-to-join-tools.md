# Cómo proponer una Tool (billetera/SDK/kit) por PR

1) Crea una ficha en `/wallets/<tool>.md` con:
   - Enlaces oficiales (sitio, docs, repo).
   - Requisitos (HTTPS, redes soportadas, etc.).
   - Snippet mínimo para conectar/firmar.
2) (Opcional) Agrega un QuickStart en `/quickstarts` si tu tool lo necesita.
3) Abre un PR usando el template y explica cómo probarlo.
